Raw Data:

This file contains the default data before running the Monte Carlo simulation and using standard deviations. It includes historical data and upcoming game data for comparison.

All data is calculated as home team minus away team so stats less than 0 favor the home team and stats less than 0 favor the away team.

Please note that not all stats are "positive", so a positive value may sometimes be a negative indicator in the modeling (e.g., turnovers).
